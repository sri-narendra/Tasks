# ⌨️ Google Tasks Clone - Keyboard Shortcuts

Boost your productivity with these built-in shortcuts:

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` / `Cmd + K` | Focus Global Search |
| `Alt + N` | Create New Board |
| `Alt + L` | Create New List (in current board) |
| `Esc` | Close Details Panel / Active Modal / Open Menus |
| `?` | Show Shortcuts Help Dialog |

---
*Note: Some shortcuts might be intercepted by browser defaults on certain operating systems.*
